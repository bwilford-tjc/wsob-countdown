# Countdown to WS of Birding

A live countdown timer to the World Series of Birding (May 9, 2026). Features 370 bird fun facts delivered via text-to-speech when you click the title.

Originally handwritten in [CodePen](https://codepen.io/bwilf9/pen/gOgjLjL/9b8c455ef19e5510e14be59114560c07), now deployed at [wsob-countdown.bwilf.app](https://wsob-countdown.bwilf.app).

## Features

- Live countdown with days, hours, minutes, and seconds
- Time segments gray out as they expire
- Click the title to hear a bird fun fact via Web Speech API
- Different speech depending on how far away the event is

## Tech

Vanilla HTML, CSS, and JavaScript. No build step, no dependencies.

## Development

Open `src/index.html` in a browser, or run locally with:

```sh
npx serve src
```
